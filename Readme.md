# Agentic AI Lab Experiments

This repository contains the laboratory experiments for the **Agentic Artificial Intelligence** course.  
The experiments focus on basic understanding and implementation of agentic AI concepts using Python.

---

## Lab Experiments

### LAB 1: Fine-Tuning BLIP for Image Captioning
This experiment focuses on fine-tuning the BLIP model to generate captions for images.

**Files:**
- `Fine_tune_BLIP_on_an_image_captioning_dataset.ipynb`
- `lab1_agenticai.pdf`

### LAB 2: Text Splitting Techniques
This experiment studies different methods of splitting text for efficient processing in agentic AI systems.

**Files:**
- `5_Levels_Of_Text_Splitting.ipynb`
